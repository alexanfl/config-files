Advanced Settings

Duplicate tab position

This setting controls where the original tab appears when you open a duplicate link.

The default behaviour immitates the expected Chrome behaviour - the original tab is moved to where the duplicate would have appeared. This is usually the position after the tab where you click the link.

An alternative is to move the original tab to last position in its window, to make it easier to find.

Allow duplicates across Windows

Starting state

Start with the extension in switched 'On' state

Some public wifi networks redirect all websites to their own webpage till you login, or accept their terms. This can cause all tabs to redirect to the same page. Since they are all dulpicates now, the extension closes all but one of them.

If you frequently use such public Wifi networks, you may set this setting to 'Start switched off'.
